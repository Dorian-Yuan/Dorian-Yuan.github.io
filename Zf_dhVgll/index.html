做一个个人博客

2020-04-09 3 min read # 技术

完全免费/完全小白也可/不需要代码

一、前期准备

1️⃣一个能用的电脑

2️⃣注册一个GitHub账号

3️⃣下载并安装Git

一直NEXT就完事，无特殊要求，下文中并不会用到它😁

4️⃣下载并安装Gridea

下载会有点慢，找我要安装包可能会比较快

5️⃣一个敢于多次试错的心

二、主要操作

1️⃣进入github点击右上角头像--settings--developer settings--personal access tokens

创建好一个新的token后，填好对应的信息（），勾选且只勾选repo相关权限，然后点击generate new token

这个token只会显示一次，请务必copy下来保存好，之后要用到的

2️⃣返回github主页，点击左侧Repositories，然后点击Creat a new repository。建议你创建仓库名为 用户名.github.io 的仓库，因为 Github 会默认将此仓库设置成Github Pages 可访问类型，若不是用此命名的仓库，则不能默认使用 Github Pages，所以请务必认真确认

3️⃣打开安装好的Gridea，点击左侧远程按钮，然后点击基础配置，如图按要求填写相关信息，令牌处填刚刚要你保存好的token，CNAME为自定义域名配置，没有可不填

4️⃣再进入github点击右上角头像--settings--developer settings--OAuth Apps，创建一个新的OAuth application，然后点击Register application 保存

5️⃣第4步结束后你会得到一个Client ID和一个Client Secret同样复制到粘贴板备用

6️⃣重回Gridea打开远程--评论配置，如图打开相关开关并填入Client ID和Client Secret等相关信息

7️⃣点击左下角的同步按钮，你的Blog就初步建成了！

三、完善你的Blog

1️⃣点击美化主题，寻找你钟意的那个，点击进去之后点击Clone or Download，选择Download Zip下载

2️⃣解压，找到打开后有四个文件的那个子文件夹(不是打开之后还是同名文件夹的文件夹)，剪切这个文件夹，然后在下图所示路径粘贴(documents也可能是汉字文档)

3️⃣然后进入Gridea点击 主题--基础配置中选择对应主题，配置完成后同步即可应用每个主题的功能不同有的自带天气插件，音乐插件，宠物挂机，友链模块，只需要在主题栏对应设置就ok了，注意每次更改完保存后再同步就可以应用到你的博客上了

4️⃣写作需要用到MARKDOWN，轻度使用的话很容易掌握，友情链接点击即可获得简单教程一份

四、总结

Gridea的配置简单易懂，但是玩法还是很多的，你大可以自己去好好琢磨
