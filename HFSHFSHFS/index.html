更好的方式代替FTP

什么是HFS？

在Windows下运行的HFS（全称：HTTP File Server）顾名思义：HTTP协议的文件服务器。

这是一个非常轻量级的文件服务器，从第一次见它就没超过3MB。正如下图所展示的，HFS不仅支持Windows，还支持Linux下的wine容器。

其次，上图所展示的内容包含了软件的另外一个特性——无需安装，原文”No need to install”很明显它可以白嫖。当然，这里也有个问题，大家都白嫖了，作者当然就不会花太多的精力来维护软件。但是对于学生党来说，很多场景下基本都够用了。

说完介绍，下面开始说怎么使用了，以及一些需要注意的地方。
——————————————————————————————————————————
##1️⃣下载
下载链接

##2️⃣整理
这个软件是直接开箱就能用的，对于一些有洁癖的人比如我来说，显然会无法忍受这种情况。
莫名其妙多个文件看起来好不舒服，而且放桌面好乱。所以，第二步需要做的是找个地方，比如C盘根目录下创建一个例如hfs的文件夹，然后将hfs.exe放进去，然后对着hfs.exe点右键发送一个快捷方式到桌面。

##3️⃣调整
第一次使用时会弹个窗口出来，大意是指是否将hfs的部分功能添加到右键菜单栏里。学生党不用在意太多，闭着眼睛选就行，或者要纠结的话就选否。

##4️⃣打开
打开后就是这样

具体的各个部分，都做了尽可能详尽的标注，简单明了。
作为一款开箱即用的软件，我们在快速上手的时候暂时只需要去了解展示文件目录树的位置和地址栏中所展示的地址。一般情况下，在网络环境比较简单的时候，我们可以直接尝试访问没有任何文件的链接，此刻的我们会看到这个画面。就是没有文件的意思

##5️⃣使用
添加文件和文件夹都是一个道理。比如，我把图里的Movie文件夹放到HFS左边的框中。

松手后会有这么一个提示框

根据提示，＂Real　folder＂更快，适用于大文件夹，＂Virtual folder＂更简洁，适用于小文件夹。别问选哪个，问就是搞快点，选＂Real　folder＂。选了之后会出现这个，就说明添加成功了。

看回地址栏，有没有发现什么变化呢？没错，当我选中文件夹时，地址栏帮我们自动补全了地址，那么直接复制到浏览器试试吧。

##6️⃣补充
移动设备上安装一个支持边下边播的浏览器例如QQ浏览器，通过浏览器访问上图中的地址，随便打开一个视频，就可以直接播放，无需下载，而且可以随意拖动进度条，演示图如下：
